# Let's go hiking


## 앱 소개

등산러들의 산 도장깨기 앱 

 등산하고싶은 주말, 등산하기 좋은 산을 찾고있나요? 

"이번주 등산고?"는 산에 대한 정보와 산행포인트,  더불어 지역별, 난이도별 산에 대한 정보를 제공해요.

이미 산을 다녀온 등산러들의 후기도 참고해보세요:)

✔ 지역별, 난이도별 산 추천

✔ 산 검색 기능

✔ 가고싶은 산 저장

✔ 산 위치 보기

✔ 한줄 등산 리뷰 공유

✔ 나만의 등산 리뷰

## 구현 기능
- 산 정보 API서비스, 북마크기능, 산 위치정보제공, 등산리뷰 채팅, 산행일기
<br />
MVVM -retrofit2, databinding, liveddata, koin , kotlin
